# Welcome to Adatree Design documentation

The Adatree UI is built using [React](https://reactjs.org/).

We use [Atomic Design](https://bradfrost.com/blog/post/atomic-web-design/) principles to help organise and maintain our components.

The design is based around [Material Design](https://material.io/) and relies heavily on the [Material UI library](https://mui.com/).
